  • In real mode, the segment base is calculated by shifting the selector value by a specific number of bits, usually 4.
  • In 64-bit mode, the segment base is often ignored and treated as 0.
    For these registers, the segment base is not expanded from 32 bits to 64 bits.
  • In 64-bit, the limit of most segments are ignored, however when the LMSLE feature is enabled, the limit value is added to xFFFFFFFF00000000 to get the actual limit.
    This is only available on certain AMD CPUs.

Intel 8089 registers

Note: The Intel 8089 was a coprocessor intended to be used with an Intel 8086/8088, somewhat similarly to the Intel 8087. Unlike the latter, the Intel 8089 architecture was never updated or incorporated into later CPU designs. For the sake of completeness, its register repertoire is included here.
